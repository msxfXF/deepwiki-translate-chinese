# 示例文档

欢迎使用 **DeepWiki** 翻译演示。

- 保持代码块不变：```python
print("Hello, world!")
```
- 保留像 [DeepSeek](https://www.deepseek.com/) 这样的链接和 ![Logo](images/logo.png) 这样的图片。
- 避免翻译文件名，如 `data/sample.csv`。
- 保持表格结构：

| 功能 | 状态 |
| --- | --- |
| 分块 | 就绪 |
| 异步 | 已启用 |

这是一段较长的文本，用于测试分块和格式化。翻译时应保持 Markdown 结构完整，同时将自然语言转换为中文，不更改 URL 或内联代码 `const value = 42`。
